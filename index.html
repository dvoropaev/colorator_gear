    • Colorator

    Colorize your text in the terminal.

    Project maintained by octopress Hosted on GitHub Pages — Theme by mattgraham

    Install

    gem install colorator

    Example

    require 'colorator'

"this string".red
# => \e[31mthis string\e[0m
"my string".blue
# => \e[34mthis string\e[0m
# etc...

    Supported Colors

    • black
    • red
    • green
    • yellow
    • blue
    • magenta
    • cyan
    • white
    • bold

    Why

    There are a bunch of gems that provide functionality like this, but none have as simple an API as this. Just call "string".color and your text will be colorized.
